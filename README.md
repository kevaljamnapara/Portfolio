## ğŸ¨ Customization

### Change Colors
Edit CSS variables in `styles/main.css`:
```css
:root {
    --primary-color: #3b82f6;      /* Change to your color */
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
}
```

### Update Personal Info
Edit `index.html`:
- Hero section: name, title, tagline
- About section: bio and highlights
- Contact section: email, links

### Add New Projects
Edit `scripts/data.js` and add to the `projects` array.

### Change Fonts
Update font-family in `styles/main.css`:
```css
--font-family: 'Your Font', sans-serif;
```
